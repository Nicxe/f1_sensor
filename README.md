## Your home, in sync with Formula 1

`F1 Sensor` is a custom [Home Assistant](https://www.home-assistant.io/) integration for Formula 1 schedules, standings, live timing, dashboards, automations, and replayed sessions.

It works without F1TV Auth. Public live timing powers core live entities such as track status, race control, weather, driver timing, tyres, and conservative incident alerts. Optional F1TV Auth can unlock extra live features such as live Track Map, Pit Stops, Championship Prediction, and earlier incident candidates when Formula 1 provides the data.

Whether you want lights to react to a Safety Car, a dashboard with live timing cards, a Track Map during sessions, or notifications for likely stopped cars, F1 Sensor keeps Home Assistant aligned with the race weekend.

## Live Data Cards

F1 Sensor includes custom Lovelace cards for live and replay dashboards, including timing, race control, tyre strategy, pit stops, investigations, championship prediction, replay controls, and Track Map.

<img alt="F1 Sensor Live Data Cards" src="https://github.com/user-attachments/assets/9ec7b6c0-9584-4647-baac-03407d73ef99" />

The card assets are bundled with the integration and registered automatically in Home Assistant. Existing dashboard card types stay the same, so you can keep your current dashboard configuration after updating.

## Key Features

- Public live timing without F1TV Auth
- Optional [F1TV Auth](https://nicxe.github.io/f1_sensor/features/f1tv-auth) for extra live timing features
- [Track Map](https://nicxe.github.io/f1_sensor/features/track-map) for live and replay car positions
- [Incident Detection](https://nicxe.github.io/f1_sensor/features/incident-detection) for likely stopped cars and on-track incidents
- [Replay Mode](https://nicxe.github.io/f1_sensor/features/replay-mode) for watching completed sessions later with Home Assistant entities
- Bundled [Live Data Cards](https://nicxe.github.io/f1_sensor/cards/cards-overview)
